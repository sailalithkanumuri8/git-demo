# Git Demo - BDBI @ Georgia Tech

This is a sample Python project for learning Git basics.

## Project Overview
A simple calculator and student grade tracker to demonstrate Git workflows.

## Getting Started
1. Clone this repository
2. Run the calculator: `python calculator.py`
3. Run the grade tracker: `python student_grades.py`

## Features
- Basic arithmetic operations
- Student grade management
- Data analysis utilities

## Git Commands to Practice
- `git add <file>` - Stage changes
- `git commit -m "message"` - Commit changes
- `git push` - Push to remote
- `git checkout -b <name>` - Create a branch, and switch to it
- `git checkout <branch>` - Switch branches
- `git merge <branch>` - Merge branches
